FastRock Image viewer
==============

Simple image viewer

Features
- Open files in full screen mode
- Large images are scaled to the size of the window
- Switching images with the mouse wheel
- Esc is exit

Requirements
- Ubuntu
- Python
- PyGTK

Installation
 - make install

Usage
- friv.py /full/path/to/file.jpg
